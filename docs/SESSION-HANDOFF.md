Verify with `gh pr view <n> --json state,mergedAt`, `git worktree list`, `git ls-remote --heads origin <branch>` (don't trust exit codes).

- **`git checkout <ref> -- src/` does NOT delete files absent from `<ref>`** — for a complete revert-proof, `rm` any NEW files explicitly, then `git checkout origin/main -- <modified existing files>`, run tests (expect failures), then `git checkout HEAD -- src/` to restore.
